# 轨迹查看器

这是一个macos的原生应用，可以查看一个人的轨迹。

## 核心逻辑

轨迹有csv和gpx两种，在exampledata下。这两个轨迹都是四年来所有的的轨迹，你可以读取前几行看看格式。但是你切不可一次把他们都读完，这会立刻爆掉你的上下文。因为是几年的轨迹，所以这些文件一定都很大，文件打开以后，首先把数据读取到sqlite中，按时间从新到旧排序。用md5区分文件，永久保存最近5次打开的数据，省的下次再花很多时间加载。

在分析轨迹数据的时候，还要加上一个数据列，日期。根据记录轨迹点的地点，来计算当地的时区，然后计算出日期。

然后根据日期和当日的所有轨迹，做出另一张数据表，日期-每日的轨迹长度，后面会在UI里展示。

## 界面设计

应用的主界面是一个地图。上方是工具栏。工具栏从左到右是：

1. “打开”，在左侧打开轨迹数据，可以打开csv或者gpx的文件。默认打开后为“一日轨迹”状态，显示最近一日的轨迹。

2. HDR开关。如果打开HDR开关，轨迹就用HDR的高亮来显示。如果关掉HDR开关，适当降低背景地图的亮度和饱和度。其实苹果地图的饱和度比较高，现在的轨迹看着有点吃力。还需要在轨迹的线条周围加一点阴影。辨识度会好些。这几个参数几种放在一个地方，方便我自己调整。在支持HDR的苹果设备上，默认打开，否则默认关闭。

2. 观察模式切换按钮，在右侧，切换一日或者多日。
3. 日历部分，在右侧。首先，你要设计一个类似github绿色砖块的日历。每个日子根据轨迹距离的长短，使用github绿色砖块的颜色进行上色，0-30km是浅色，80km是中等绿色，300km是最深的绿色，不要简单地在0-300设置线形渐变，而是在距离较少的区间里变化明显一点，后面长距离的变化不那么明显，但这个颜色的选择要是一个平滑的渐变。
(a)在一日模式下，这里是一个单行的迷你的日历，一共有13个日期，每个日期是一个方格，顶部是中文的“日一二三四五六”，用来标识星期几。左上角是小字的月份，右下角是大字的日，迷你日历的每个日期都要使用上述的“github绿色砖块”算法上色。最中间是当前查看的日期，左边是前6日，右边是后6日。如果前后没有那么多日期，则留空。如果有些日子没有数据，则跳过。然后还有一个按钮可以展开日历，点击就可以展开日历，打开一个竖着滚动的月历窗格，窗口大小能展示三个月的月历，月历开始和结束的日期为用户轨迹中最旧一天和最新一天。用户滚动月历，选择一个日期。月历中的每个日期都使用“github绿色砖块算法”上色，如果有些日子没有足迹数据，则不光不上绿色，还要把日期显示为灰色，不可选中。
(b)在多日模式下，这里显示日期范围。点击日期范围，可以展开日历，打开一个竖着滚动的月历窗格，窗口大小能展示三个月的月历。选择日期范围，月历中的每个日期都使用“github绿色砖块算法”上色，如果有些日子没有足迹数据，则不光不上绿色，还要把日期显示为灰色。没有足迹的日子可以被选中，作为开始结束或者区间内的日子。在月历中选择时间范围，如同agoda，booking中选择住宿日期的那个月历一样，选择一个开始时间，选择一个结束时间，就限定了时间范围。这里要参考agoda/bookings的日历选择器的动画，清楚地展示日期范围。

除了顶部的工具栏，下面就是一张大地图了。地图用苹果系统的地图api。要把所有应当展示的轨迹点展示在上面，并且按照时间顺序，用平滑的曲线连接（这里需要选择一个平滑算法，单独列成一个函数方便后期修改）。在没有选择日期的时候，这里缩放小最小，显示世界地图，选择了日期则平移和缩放到整个轨迹占满70%的面积。

地图上的轨迹展示有两个问题需要注意：

1、严格设定一个中国大陆的边界线，因为有个著名的“火星坐标偏移”问题，如果你不修正坐标，看到的所有轨迹都会偏移。

2、长距离国际航线在地球球面投影问题，当乘坐国际航班时，因为地球是圆形，所以在咱们的2d平面图上的投影应该是一条弧线。如果轨迹中两点长度大于200km，这个问题就需要被考虑在内。

对于一日模式，用彩虹色，红橙黄绿蓝靛紫，平滑地给曲线和点上色，象征一天从开始到结束，红色为这天的第一个时间点，紫色为这天的最后一个时间点。配色用pastel colors色系。鼠标移动到某一个轨迹点上的时候，有一个小气泡能显示那个点的经纬度、高程、时间。这个浮窗应当是有一个指向轨迹点的三角，并且黏在轨迹点上，而不是跟着鼠标跑。此时轨迹点也要稍微长大一点。

对于多日模式，用彩虹色，红橙黄绿蓝靛紫，离散地为每一天指定一种颜色。红色代表第一天，紫色代表最后一天。每一天不展示任何的轨迹点，而是一段平滑的轨迹。鼠标移动到轨迹上的时候，展示日期和这天的总距离。点击这天的轨迹，可以暂时切换到一日模式，展示这天的轨迹。但是在“一日/多日”切换按钮的左侧增加一个按钮，返回刚刚的多日模式。

程序启动的时候默认加载上次的轨迹文件，上次的日期设置。
